# Code Connect — Versão Original

Esta branch contém a versão original do projeto, desenvolvida durante curso da Alura.

O código encontra-se estruturado de forma monolítica, com a lógica centralizada em um único arquivo (`index.js`) e nomenclatura em português.

## 📌 Características da versão original

- Estrutura não modular
- Lógica concentrada em um único arquivo JavaScript
- Manipulação direta do DOM
- Organização inicial proposta pelo curso

## 🔄 Evolução

Este projeto passou posteriormente por um processo completo de refatoração, incluindo:

- Modularização com ES Modules
- Separação de responsabilidades
- Padronização da nomenclatura para inglês
- Organização em camadas (controllers, dom, events, modules e services)
